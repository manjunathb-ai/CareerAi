# 🚀 CareerAI — AI Powered Career Assistant

**CareerAI** is a full-stack AI SaaS platform that helps users accelerate their career using intelligent automation and voice interaction.

---

## 🧠 Features

### ✨ AI Resume Builder

* Generate ATS-friendly resumes
* Smart content suggestions
* Downloadable format

### 🌐 Portfolio Generator

* Create modern developer portfolios
* AI-generated content
* Ready-to-deploy layouts

### 🎤 AI Interview Coach

* Practice mock interviews
* Real-time feedback
* Role-based questions

### 🔊 Voice AI Assistant

* Talk to AI using voice
* Real-time responses
* Natural conversation flow

### 💳 SaaS System

* User authentication (Supabase)
* Subscription system (Stripe)
* Dashboard for users

### 💰 Affiliate Monetization

* Smart recommendations
* Context-based affiliate links

---

## ⚙️ Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Framer Motion
* Google Gemini API
* Supabase (Auth + DB)
* Stripe (Payments)
* Vercel (Deployment)

---

## 🧩 Architecture

```
Frontend → API → AI Agent → Modules

/api/agent       → AI routing
/api/resume      → Resume generator
/api/portfolio   → Portfolio generator
/api/interview   → Interview coach
/api/feedback    → User feedback
```

## 🚀 Getting Started

### 1. Clone repo

```bash
git clone https://github.com/yourusername/career-ai.git
cd career-ai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Add environment variables

Create `.env.local`:

```env
GEMINI_API_KEY=your_key
SUPABASE_URL=your_url
SUPABASE_ANON_KEY=your_key
STRIPE_SECRET_KEY=your_key
```

### 4. Run project

```bash
npm run dev
```
